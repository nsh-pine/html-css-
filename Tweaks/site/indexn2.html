  • Lecture 7 was about listing.
    For unordered list, you have to use <ul></ul> tag and you have to use <li></li> in the "ul" tag or else it won't work. It's <ol&></ol> for ordered list.

  • Lecture 8 was about character entities.
    It is about tags to use when you want to display certain characters that browser might render as part of the html tags.
    For example, if you want to display less than character "<", you have to type &lt; and as well &gt; for ">.quot;
    If you want to show "&", you have to type &amp;.
    Check this page(Entities) and it's source for reference.

  • Lecture 9 was about creating links.
    You link another page with <a href="link.html">Write something here</a>.
    You can write target="_blank" in anchor(a) tag if you want the browser to open the link in a new tag or window.
    View this page(Links) and it's source for more reference.

  • Lecture 10 was about putting images in html pages.
    <img src="blahblah.jpg"> to link an image to the page. You can also link images from other websites.
    It is good practice to define the dimensions of the image when you put up an image on your page.
    For example, <img src="https://i.kym-cdn.com/entries/icons/original/000/031/406/19ylf.jpg" height="256" width="411"> will bring you the following image.

    Because you defined the dimensions of the image, even though the link to the image is broken, the area for the image is reserved.

    You can also add alt="jahy-sama-smiling" to the "img" tag to describe the image if the link is broken or for people with visual impairment.

    View this page(Images) and it's source for reference.
